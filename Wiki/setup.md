## Project Setup

### Requirements

- Python
- Flask (python)
- venv

### Running the API

- Setup a python `venv`.
- Install python Flask.
- Set `FLASK_APP` env var to point at `./Source/Adapter/flask_app.py`.
- Run `./venv/Scripts/python -m flask run`.
- Alternatively, run `./runFlaskServer.ps1` which sets the env var and runs the
flask command above.
